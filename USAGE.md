# 使用指南

## 环境要求

- Python 3.11+
- Node.js 18+
- npm

## 快速启动

### 1. 启动后端服务

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

服务默认运行在 `http://127.0.0.1:8000`

在线 API 文档：`http://127.0.0.1:8000/docs`

### 2. 启动前端服务

新开一个终端：

```bash
cd web
npm install
npm run dev
```

服务默认运行在 `http://127.0.0.1:5173`

前端开发服务器已配置代理，`/api` 请求自动转发到后端。

## 使用流程

```
注册账号 → 创建班级 → 添加学生 → 管理档案
```

### 第一步：注册教师账号

1. 浏览器打开 `http://127.0.0.1:5173`
2. 点击 **"立即注册"**
3. 填写用户名、姓名、密码（至少 6 位）、联系电话（选填）
4. 提交后自动登录跳转到控制台

### 第二步：创建班级

1. 在控制台顶部点击 **"新建班级"**
2. 输入班级名称（如：三年级一班）和年级（如：三年级）
3. 确定后班级自动选中

### 第三步：添加学生

#### 方式一：从控制台添加
- 在控制台点击 **"添加学生"** 按钮
- 填写学生信息：学号、姓名、性别、出生日期、联系方式等
- 选择所属班级
- 提交

#### 方式二：从学生管理页添加
- 左侧导航进入 **"学生管理"**
- 点击右上角 **"添加学生"**

### 第四步：查看与管理学生

#### 学生列表
- 控制台显示当前班级学生卡片
- 学生管理页支持按姓名/学号搜索
- 点击学生卡片进入详情页

#### 学生详情
- 查看学生完整信息
- 点击 **"编辑"** 修改信息
- 点击 **"档案记录"** 查看档案时间线
- 点击 **"删除"** 移除学生（不可恢复）

### 第五步：档案记录

1. 在学生详情页点击 **"档案记录"**
2. 在表单中选择记录类型（学业/行为/健康/其他）
3. 填写标题和内容
4. 提交后自动归档到时间线

## 常见操作

| 操作 | 路径 |
|------|------|
| 切换班级 | 页面顶部班级下拉框 |
| 搜索学生 | 学生管理页搜索框 |
| 查看统计 | 控制台仪表盘 |
| 新建班级 | 控制台顶部 / 班级下拉框旁 |

## 注意事项

- 学号在学生系统中唯一，不可重复
- 删除学生操作不可恢复
- 档案记录按创建时间倒序排列
- 开发环境使用 SQLite 数据库（`backend/student_ms.db`），重启服务数据不丢失
